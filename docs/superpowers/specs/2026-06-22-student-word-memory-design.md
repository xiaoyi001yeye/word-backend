# 学生全局卡片盒与错词收藏设计

## 目标

在现有学生工作台基础上增加全局单词记忆状态，让计划内学习结果沉淀为学生自己的长期卡片盒，并开放错词本与收藏夹。计划内进度继续由 `StudyWordProgress` 负责，全局记忆不替代老师布置计划的完成率。

## 范围

- 新增 `StudentWordMemory`：按 `studentId + metaWordId` 唯一保存盒位、掌握度、下次复习日、累计正误、连续答对、自动错词和收藏状态。
- 新增 `StudentWordMemoryEvent`：记录每次影响全局记忆的计划学习事件，保留来源、结果和盒位变化。
- 今日学习提交后同步更新全局记忆。
- 学生可查看自己的全局记忆、错词本、收藏夹。
- 学生可收藏或取消收藏自己直接分配或班级分配词书中可见的单词。
- 前端词库页点亮“错词本”和“收藏”页签，并在单词详情中提供收藏切换。

## 延期

- 自由学习队列。
- 学生自测与考试分轨。
- 30 天统计图表。
- 手动移出自动错词本。

## 记忆规则

- `CORRECT`：盒位最多提升 1，连续答对加 1，累计答对加 1。
- `INCORRECT`：盒位下降到低位，连续答对清零，累计答错加 1，`autoWrong=true`。
- `SKIPPED`：长期记忆按错误处理，`autoWrong=true`；计划内仍按现有规则完成当天任务。
- 盒位复习间隔为 `[0, 1, 2, 4, 7, 15, 30]` 天。
- 盒位达到 5 且连续答对至少 3 次时，`autoWrong=false`。
- 掌握度为累计答对次数 / 总次数的百分比。

## 接口

- `GET /api/students/me/word-memory`
- `GET /api/students/me/wrong-words`
- `GET /api/students/me/favorite-words`
- `PATCH /api/students/me/word-memory/{metaWordId}/favorite`

列表返回单词基础信息、音标、释义、音节详情和记忆状态。收藏接口请求体为 `{ "favorite": true | false }`。

## 权限

- 所有接口只允许 `STUDENT` 调用。
- 学生只能访问自己的记忆状态。
- 收藏只能操作当前学生直接分配或班级分配词书中存在的单词。

## 验收

- 今日学习答错或跳过后，该词进入错词本。
- 今日学习答对会更新盒位和掌握度。
- 连续答对满足阈值后自动错词状态解除。
- 收藏和取消收藏立即反映到收藏夹。
- 取消收藏不影响自动错词状态。
- 直接分配或班级分配词书中的单词均可收藏。
- 未通过直接分配或班级分配获得的词书单词不能被收藏。
- 前端词库页可切换“我的词书 / 错词本 / 收藏”。
